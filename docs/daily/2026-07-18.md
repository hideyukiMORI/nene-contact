## ★教訓

- **codemod は move するが import を書換えない**（`rewrite 0`・特に `@/` エイリアス import）→ 生出力と手修正を 2 コミット分離で gap 可視化（fleet#83）。
- **required status check 施錠下で required context の job を rename すると PR が永久 BLOCK**（旧 context が Expected のまま）。回避＝required を出す job 名は据え置き、別 job を distinct 新名で additive 追加→hub が required 追加。
- **クロスオリジン webfont は CORS(ACAO) が要る**。自前ホストのフォントを別オリジンのページで使うと、CSP でなく ACAO 不在でブロック→フォールバック。切り分けは「CSP ヘッダ」でなく「font レスポンスの ACAO」を見る。
- **ライブ埋め込みのハッシュ直参照は再デプロイで壊れる**。不変エイリアス（`/embed/embed.js`・SRI なし）へ寄せてから再デプロイするのが安全順序。
