# Glossary

Plain-language meanings. **Spelling of identifiers** is defined in [`terminology.md`](./terminology.md).

| Term | Meaning |
| --- | --- |
| **Contact form** | Operator-defined set of fields and notification rules |
| **Public form key** | Opaque id embedded in `data-form`; not the internal database id |
| **Submission** | One visitor POST captured as a record |
| **Handoff** | HTTP call that creates or links a resource in a sibling product |
| **Embed** | `embed.js` snippet and hosted UI loaded on a third-party site |
| **Allowed origins** | Hostnames permitted to POST submissions for a form |
