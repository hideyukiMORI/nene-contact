# Roadmap

NeNe Contact — embeddable contact forms on NENE2.

## Phase 0: Governance ✅ (in progress)

Charter, ADRs 0001–0010, scope contract, embed spec, privacy rules, terminology, sibling integration drafts.

## Phase 1: Runtime foundation

Multi-tenant org/auth, `GET /health`, OpenAPI skeleton, CI, admin JWT.

## Phase 2: Forms + embed MVP

Contact form CRUD, public schema/submit, embed.js (floating), inbox list/detail, email notification.

## Phase 3: Channels + handoff

Slack/Chatwork, webhooks, Deal opportunity handoff, Vault attachment handoff.

## Phase 4: Sibling depth

Invoice draft client, Records select options, Concierge ingest API, MCP write with audit.

## Not on roadmap

- Chat scenarios (Concierge)
- Quotes, invoices, reconciliation (Invoice / Clear)
- Bank CSV (Profile)
- Full CRM (Deal remains pipeline SSOT)
